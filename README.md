# Scrumer PM

An opensource AI-powered project management assistant for software teams.

## About The Project

Scrumer PM is a modern, AI-driven project management tool designed to streamline the software development lifecycle. It integrates with GitHub and leverages Google's Gemini AI to provide intelligent insights, automate tasks, and enhance team collaboration.

Key features include:
- A real-time, interactive project board
- Integrated backlog management
- AI-powered analytics and reporting
- Seamless GitHub integration for issue tracking and code management
- An innovative "Github Hub" for direct interaction with repositories

This project is built with React, TypeScript, and Vite, offering a fast and responsive user experience.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js and yarn
- A `.env.local` file with your Gemini API key:
  ```
  VITE_GEMINI_API_KEY=YOUR_API_KEY
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username/scrumer-pm.git
   ```
2. Install YARN packages
   ```sh
   yarn install
   ```

### Running the Application

To start the development server, run:
```sh
yarn dev
```
This will start the application on `http://localhost:5173`.

## Features

- **Project Board:** Visualize and manage your team's workflow with a drag-and-drop Kanban-style board.
- **Backlog:** A dedicated space to prioritize and manage all upcoming tasks and user stories.
- **Analytics:** Gain insights into your project's progress with data-driven analytics.
- **GitHub Integration:** A dedicated hub to connect with GitHub, manage issues, and view repository data.
- **AI-Powered:** Utilizes the Gemini API for intelligent features and suggestions.
- **Settings:** Customize the application to fit your team's needs.
